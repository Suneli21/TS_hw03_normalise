# TS_hw03_normalise

> Выполнила Гусманова Э.Э., ИИ-25


### Основная гипотеза

Нормализация временных рядов оказывает существенно большее влияние на качество глобальных нейронных моделей, чем на классические статистические модели или модели на основе деревьев (tree-based). Дополнительно исследуется, как разные методы масштабирования влияют на качество прогнозов.

---

## Данные

В работе используется поднабор данных из M4 Competition:

* 150 временных рядов
* одномерные ряды различной природы (экономические, демографические, финансовые и др.)
* фиксированная длина каждого ряда
* разбиение:
  * train
  * test (горизонт прогнозирования: 18 точек)

### Описание файлов

#### `utils/`

Вспомогательные модули проекта.

* `experiments.py`
  
Реализация экспериментов: обучение различных моделей с применением нормализаций, подсчёт метрик и прогнозирование на тесте.

* `metrics.py`
  
Метрики качества прогнозирования.

* `preprocessing.py`
  
Предобработка данных: разделение на тренировочную и тестовые выборки, Box-Cox, нормализация и т.д.

---

#### `TS_hw3_Gusmanova_EE.ipynb`

Основной ноутбук с исследованием:

#### `TS_project_report.pdf`

Отчёт с краткой сводкой по проведённой работе: гипотеза, описание данных, метрики, стратегии нормализации, модели предсказания и т.д.

## Установка и запуск

1. Клонирование репозитория

```bash
git clone https://github.com/Suneli21/TS_hw03_normalise
cd TS_hw03_normalise
```

2. Создание виртуального окружения

```bash
python -m venv venv
```

Активация:

* Windows:

```bash
venv\Scripts\activate
```

* Linux / MacOS:

```bash
source venv/bin/activate
```

3. Установка зависимостей

```bash
pip install -r requirements.txt
```

4. Запуск ноутбука

```bash
jupyter notebook
```
Откройте файл TS_hw3_Gusmanova_EE.ipynb.
